## Maven
How to include the API with Maven:


```html
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>

<dependency>
  <groupId>com.github.Warsteiner37</groupId>
    <artifactId>UltimateJobs</artifactId>
    <version>SNAPSHOT</version>
  <scope>provided</scope>
</dependency>
```
